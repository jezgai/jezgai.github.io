Axis Mundi Tierra Media - Ayuda para crear personajes

Con el fin de no tener páginas duplicadas, he eliminado de aquí la creación de personajes con las clases pensadas para Tierra Media.

Para poder crear usuarios con las cutre-clases que monté para la Tierra Media, lo primero que hay que hacer es descargar el archivo json de las Clases para Tierra Media, abrir la página de Ayuda para crear personajes de Axis Mundi, pulsar el botón "Carga Clases", marchar el check "Sustituir clases por fichero", seleccionar el json descargado y finalmente pulsar sobre "Cargar".
